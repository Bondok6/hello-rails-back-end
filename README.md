## Getting Started

### Clone this repository

```bash
git@github.com:bodnok6/hello-react-rails.git
cd hello-react-rails
```
### Install gems
```bash
bundle install
```

### Install dependencies
```bash
npm install
```

### Setup database
```bash
rails db:create
rails db:migrate
rails db:seeds
```

### Start server
```bash
rails s
```
